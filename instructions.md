# StigViewer CFML Installation

StigViewer CFML is a CommandBox module that adds the `stigviewer`
command to the CommandBox CLI.

## Prerequisites

You must have CommandBox installed.

Download and install CommandBox from:
https://www.ortussolutions.com/products/commandbox

After installation, verify:

    box version

------------------------------------------------------------------------

## Install

Install the module from ForgeBox:

    box install stigviewer-cfml@ccsimmons

Or from inside the CommandBox interactive shell:

    install stigviewer-cfml@ccsimmons

------------------------------------------------------------------------

## Verify Installation

Run:

    stigviewer about

If the command prints usage information, the module is installed
correctly.

------------------------------------------------------------------------

## Updating

To update to the latest version:

    box update stigviewer-cfml@ccsimmons

------------------------------------------------------------------------

## Uninstall

To remove the module:

    box uninstall stigviewer-cfml@ccsimmons
